API Project: URL Shortener Microservice

User Story:

  1. To create a URL Shortner Microservice. Given a URL as a parameter, a JSON object will be returned with the shortened url link.
  2. Clicking on the newly created shortened url will redirect you to the original link that was input.
  3. If an Invalid url is entered, the JSON response will contain an error instead.

Short URL Creation

Example Usage:

  • Post https://www.freecodecamp.com bove will return this JSON response: {"original_url":"https://www.freecodecamp.com","short_url":1}
  • https://bevel-scissor.glitch.me/api/shorturl/2 will redirect to: https://www.google.com
  • If an Invalid url is entered, the JSON response will be: {"error":"invalid URL"}
